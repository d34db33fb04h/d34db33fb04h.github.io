Retro (Windows Easy)

Retro is a box working around pre-created windows 2000 machines. Pre-creating a computer means adding a computer to AD without using it to join a host to the domain right away, it just gets used later. There is a ‚ÄúPre-Windows 2000‚Äù compatibility option that can be selected when creating a computer from ADUC, still present in Windows Server 2022. A computer created with this option will have a password equal to the computers name in lowercase without the ‚Äò$‚Äô. This allows you to look at ADCS templates being used that are vulnerable, which leads to privilege escalation to Administrator.
